# VA-demo-project  
The task is to enter data from a PDF into a spreadsheet (Excel .CSV or .XLSX). The required fields are:

- Name  - Title  - Company  - Street Address  - City  - State  - ZIP  - Phone  - Fax  - Email  - Website (if available).
